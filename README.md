# Billionaires Boulevard — Bio Scan Route

> **Fan-made preservation project. All route data belongs to the original creators of CMDR's Toolbox.**

---

## About

This is an interactive checklist for the **Billionaires Boulevard** biological scanning route in Elite Dangerous, originally published on **CMDR's Toolbox** (`cmdrs-toolbox.com`).

CMDR's Toolbox has since been sunset. This page was created to preserve and openly share the route data with the Elite Dangerous community, so it doesn't get lost.

**I am not the original author of this data.** Full credit goes to the developers and contributors behind CMDR's Toolbox for compiling and publishing this route.
